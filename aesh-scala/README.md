aesh-scala
==========

####How to run:

You will need sbt (http://www.scala-sbt.org) installed.

```
$ git clone https://github.com/aeshell/examples.git
$ cd examples/aesh-scala
$ sbt run
```

Commands available:

- cat
- cd
- clear
- echo
- harlem
- less
- ls
- matrix
- mkdir
- more
- pwd
- rm
- touch
