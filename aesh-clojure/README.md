aesh-clojure
============

####How to run:

You will need JDK 8 and Leiningen (leiningen.org) installed and configured.

```
$ git clone https://github.com/aeshell/examples.git
$ cd examples/aesh-clojure
$ lein run
```

Commands available:

- cat
- cd
- clear
- echo
- harlem
- less
- ls
- matrix
- mkdir
- more
- pwd
- rm
- touch
