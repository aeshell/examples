aesh-hello-world
================

####How to run:

```
$ git clone https://github.com/aeshell/examples.git
$ cd examples/aesh-hello-world
$ ./gradlew run
```

#####or

```
$ git clone https://github.com/aeshell/examples.git
$ cd examples/aesh-hello-world
$ ./gradlew clean distZip
$ cd build/distributions/
$ unzip aesh-hello-world.zip
$ cd aesh-hello-world/bin
$ ./aesh-hello-world
```
