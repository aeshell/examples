aesh-groovy
===========

####How to run:

```
$ git clone https://github.com/aeshell/examples.git
$ cd examples/aesh-groovy
$ ./gradlew clean distZip
$ cd build/distributions/
$ unzip aesh-groovy.zip
$ cd aesh-groovy/bin
$ ./aesh-groovy
```

Commands available:

- cat
- cd
- clear
- echo
- harlem
- less
- ls
- matrix
- mkdir
- more
- pwd
- rm
- touch
